# 天堂M 吃王小幫手 — 完整版 (GitHub-ready)

這個專案為完整可部署版本，包含：
- 記錄 boss 死亡時間（指令）
- 計算下次重生並排程提醒（會推播到綁定的聊天室）
- 使用 FastAPI、LINE Messaging API、APScheduler
- 不包含任何私密金鑰，請在部署平台(例如 Railway)填入環境變數

## 指令
- `王表`：顯示所有 boss 與 CD（小時）
- `王名 時間`：記錄死亡時間（例如：`巨蟻女王 13:20`）。時間可省略（則視為現在）
- `查 王名`：查詢該王的預計重生時間
- `綁定`：綁定本聊天室為提醒目標
- `解除綁定`：解除綁定
